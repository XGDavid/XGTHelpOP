# XGTHelpOP

## Description


This is a PocketMine Plugin.
With this plugin you can ask for the help of an OP, as well as to see OPs online.

## How to install

1. Install plugin

2. Stop/start server

3. Edit config

4. Restart server

5. Have FUN!


## Features
- [X] Config

- [X] Costom Message

- [X] Delay Command
